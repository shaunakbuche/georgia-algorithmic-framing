# data/processed/README.md

## Processed Data Files

This directory contains analysis-ready datasets generated by the pipeline.
These files **are tracked in git** (county-level aggregates; no individual-level data).

---

## Files

### `panel_dataset.csv` — Main analysis panel
The final merged dataset used in all regressions.
- **Unit:** County × election year
- **N:** Up to 636 (159 counties × 4 elections)

| Column | Type | Description |
|--------|------|-------------|
| `county_fips` | string | 5-digit FIPS code |
| `county_name` | string | County name |
| `year` | int | Election year {2018, 2020, 2022, 2024} |
| `turnout_pct` | float | Ballots cast / registered voters × 100 |
| `rep_margin_pct` | float | (Rep − Dem) / Total votes × 100 |
| `sentiment_index` | float | Impression-weighted mean ad polarity |
| `topic_social_share` | float | Immigration + Social Issues topic share |
| `topic_health_share` | float | Healthcare/COVID topic share |
| `topic_election_share` | float | Election Integrity topic share |
| `median_income` | float | Median household income ($) |
| `pct_bachelors` | float | % adults 25+ with BA degree |
| `population` | int | Total county population |
| `prior_turnout` | float | Turnout in previous election |
| `prior_margin` | float | Margin in previous election |
| `is_presidential` | int | 1 if presidential year, 0 if midterm |

---

### `ad_sentiment.csv` — Ad-level sentiment scores
One row per ad × county attribution (long format).

| Column | Description |
|--------|-------------|
| `ad_id` | Meta Ad Library unique identifier |
| `county_fips` | Attributed county |
| `year` | Election year |
| `polarity` | Sentiment score (VADER compound or TextBlob polarity) |
| `sentiment_method` | "vader" or "textblob" |
| `statewide_allocation` | True if impressions allocated proportionally (not targeted) |

One row per ad × county; includes dominant topic from LDA model.

### `county_topics.csv` — County-year topic prevalence

## Codebook Notes

**Sentiment index:** Positive values = more positive advertising environment.
Negative values = more negative/attack advertising. Approximately 0 = neutral.

**rep_margin_pct:** Positive = Republican advantage; negative = Democratic advantage.
Ranges roughly from −60 (heavily Democratic urban) to +80 (rural Republican).

**statewide_allocation flag:** Ads marked `statewide_allocation=True` were
distributed proportionally by county population (not by targeting data).
This introduces measurement error — see paper §6 and sensitivity analysis
in robustness checks. Approximately 40–60% of impressions are statewide-allocated.
